huddleUp

March 2023

huddleUp is a social media platform created for the 2023 SteelHacks Hackathon.

It was developed using Python Flask and SQL.

Perfect for college campuses such as the University of Pittsburgh, HuddleUp allows users to create and view events in their area and meet new people with similar interests.

KeyFinder

February 2023

KeyFinder is a Python Flask that pulls keywords from user input and displays them as keycards, complete with definitions.

KeyFinder uses the OpenAI API to decide which keyword to pull and Google Knowledge Bank Search API to find definitions in the context of the text.

September 2022
